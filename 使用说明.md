# 🚀 一键启动使用说明

## Windows用户

### 第1步：下载项目
访问 https://github.com/Apple-pie-pie/Tushare
- 点击绿色 **Code** 按钮
- 选择 **Download ZIP**
- 解压到任意文件夹

### 第2步：双击启动
1. **首次运行**：双击 `start.bat`
   - 会自动创建 `.env` 配置文件
   - 提示你编辑Token
   
2. **编辑Token**：
   - 右键点击 `.env` → 用记事本打开
   - 修改 `TUSHARE_TOKEN=your_token_here`
   - 改成你的真实Token（从 https://tushare.pro 获取）
   - 保存文件

3. **再次双击 `start.bat`**：
   - 自动安装依赖
   - 自动初始化数据库
   - 自动启动应用
   - 浏览器自动打开 http://localhost:8501

### 停止服务
在黑色命令窗口按 `Ctrl+C`

---

## Mac/Linux用户

### 第1步：下载项目
```bash
git clone https://github.com/Apple-pie-pie/Tushare.git
cd Tushare
```

### 第2步：双击启动
1. **首次运行**：双击 `start.sh`（或在终端运行 `./start.sh`）
   - 会自动创建 `.env` 配置文件
   - 提示你编辑Token

2. **编辑Token**：
   ```bash
   nano .env
   # 修改 TUSHARE_TOKEN=your_token_here
   # 按 Ctrl+O 保存，Ctrl+X 退出
   ```

3. **再次运行 `./start.sh`**：
   - 自动创建虚拟环境
   - 自动安装依赖
   - 自动初始化数据库
   - 自动启动应用

### 停止服务
按 `Ctrl+C`

## ⚠️ 常见问题

### Q1: 双击没反应？
**Windows**：确保已安装Python 3.8+（下载：https://www.python.org/downloads/）

**Mac/Linux**：终端运行 `chmod +x start.sh`，然后再双击

### Q2: 提示"未找到Token"？
编辑 `.env` 文件，把 `your_token_here` 改成你的真实Token

### Q3: 如何获取Token？
访问 https://tushare.pro → 注册登录 → 个人中心 → 接口TOKEN

### Q4: 需要联网吗？
- **首次运行**：需要联网安装依赖
- **正常使用**：需要联网调用Tushare API
- **数据查看**：可以离线查看已下载的数据

## 📱 启动后做什么？

浏览器打开后会看到4个页面：

1. **🏠 Market Terminal**：查看股票K线图
2. **🔧 Data Studio**：拉取数据（点这里开始）
3. **📊 API Catalog**：查看可用接口
4. **⚙️ Settings**：探测权限

**建议首次使用流程**：
1. 进入 Settings → 点击"探测接口能力"
2. 进入 Data Studio → 拉取交易日历和股票列表
3. 选择最近5天 → 批量拉取行情
4. 构建daily_panel
5. 回到 Market Terminal 查看数据
